# Lancer le projet

Pour garantir d'avoir les mêmes packages entre nous, il faut lancer les commandes suivantes lors de la création de l'environnement virtuel.

```bash
python3 -m venv .venv

source .venv/bin/activate

pip install -r requirements.txt
```
